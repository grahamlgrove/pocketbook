This is pain that occurs due to direct damage to the peripheral or central nervous system (e.g. due to a tumour invading the nerve).

Common characteristics include shooting, stabbing or burning style pain. Allodynia (pain on light touch) is sometimes also present.

If neuropathic-sounding pain is not responding to simple analgesia, consider adding an adjuvant agent:

Common first-line agent

  • Pregabalin

Common second-line approach:

  • If partial response to first-line, add a second agent.
  • If first line brought no benefit, switch to another agent
Pregabalin

Starting dose: 75mg at night

Titrate if necessary: up to a maximum of 300mg twice daily over 2-4 weeks

In the frail or elderly, start at a lower dose of 25mg at night and increase more cautiously

Use very cautiously if renal impairment and with reduced dose
Duloxetine

Starting dose: 30mg at night

Titrate if necessary: up to 60mg over 2-4 weeks
Valproate

Starting dose: 100mg twice daily (Or 200mg once a day, at night)

Titrate if necessary: up to a maximum of 600mg twice daily over 2-4 weeks
Other options

Amitriptyline (start at 10mg at night)

Nortriptyline (start at 10mg at night)

 

If pain control remains poor, refer to the palliative care team. Specialist options include:

  • Methadone via CSCI or orally - see Appendix - Methadone
  • Lidoocaine via CSCI - see Appendix - Lidocaine
  • Ketamine via CSCI or orally - see Appendix - Ketamine
  • Clonidine subcutaneously - see Appendix - Clonidine
