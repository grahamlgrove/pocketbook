This is pain that occurs due to direct damage to the peripheral or central nervous system (e.g. due to a tumour invading the nerve). Common characteristics include shooting, stabbing or burning style pain. Allodynia (pain on light touch) is sometimes also present.

If neuropathic-sounding pain is not responding to simple analgesia, consider adding an adjuvant agent:

Common first-line agent: Pregabalin

Common second-line approach:

If partial response to first-line, add a second agent.

If first line brought no benefit, switch to the other agent. 

 

Pregabalin

Starting dose: 75mg at night

Titrate if necessary: up to a maximum of 300mg twice daily over 2-4 weeks (less if frail or renal impairment)

Duloxetine

Starting dose: 15mg at night

Titrate if necessary: up to 60mg over 2-4 weeks

Valproate

Starting dose: 100mg twice daily

Titrate if necessary: up to a maximum of 500mg twice daily over 2-4 weeks
Other options

Amitriptyline

Nortriptyline

 

If pain control remains poor, refer to the palliative care team. Specialist options include:

  • Methadone via CSCI or orally - see Appendix - Methadone
  • Lidoocaine via CSCI - see Appendix - Lidocaine
  • Ketamine via CSCI or orally - see Appendix - Ketamine
