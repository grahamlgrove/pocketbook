• All opioids have similar efficacy and side-effects
  • Morphine is a good first choice except in renal failure or if a patch is preferable
Renal impairment Opioid choice
Normal Morphine
Mild or moderate Hydromorphone
Severe Hydromorphone or fentanyl (non-PBS)

 

  • Start with low doses and titrate up as needed using both a regular long acting opioid and a PRN immediate release formulation (see below). Initial doses are based on age and frailty
For young adults

Immediate release morphine (either hydrochloride or sulphate) 5-10mg 2-hourly PRN

+/- Modified release morphine 10mg twice daily
For elderly, frail adults

Immediate release morphine (either hydrochloride or sulphate) 2.5-5mg 2-hourly PRN

+/- Modified release morphine 5mg twice daily
