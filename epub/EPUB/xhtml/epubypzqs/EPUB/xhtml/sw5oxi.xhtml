Anticipate constipation, especially in patients prescribed opioids, and start prophylactic laxatives where appropriate. If appropriate encourage increased fluids and fruit intake and encourage mobilization if appropriate. Avoid ispaghula (e.g. Fybogel) as it worsens constipation if fluid intake is inadequate.

Initial therapy -> oral therapy with a softener and/or stimulant is appropriate

  • Docusate 1 tablet twice daily regularly (or PRN) (a softener for hard stools)
    AND/OR
  • Sennoside 1 tablet twice daily (or PRN) or picosulfate oral 10 drops twice daily (a stimulant)

Doses can be titrated up or down based on level of constipation. Titrate up the softener if stool hard and/or colicky discomfort present; titrate stimulants up if patient struggling to expel soft stool.

If constipation continues -> consider rectal treatments, e.g.

  • Bisacodyl 1 suppository daily PRN (if stools soft)
  • Microlax 1 enema daily PRN (if stool hard)

For severe, refractory constipation -> options include

  • High dose softener (PicoPrep)
  • High bowel intervention, e.g. Fleet enema
  • Methylnaltrexone (streamlined authority) if opioid-induced
  • Manual evacuation (ensure PRN analgesia / sedation available)
  • Consider - is bowel obstruction present?
  • If pseudo-obstruction, consider neostigmine 0.5mg subcut twice daily (see Appendix - Neostigmine)
Methylnaltrexone
Peripherally acting opioid antagonist
Used for opioid-induced constipation refractory to appropriately titrated  laxatives and enemas

Dose:

  • 12mg subcut (if > 62 kg)
  • OR 8mg subcut (if 38-61kg)
  • OR 0.15mg/kg (if <38kg)
  • Repeat on day 2 if required
  • If required, can continue to give every second day

Can result in a massive bowel action within an hour. Is a commode required?
Can cause severe abdominal cramps. Ensure subcut hyoscine butylbromide (Buscopan) + subcut opioids available
Contraindicated in bowel obstruction
Dose reduction required in renal failure

 

In constipation associated with spinal cord compression ->

  • Oral laxatives
    PLUS
    Bisacodyl suppository each morning (or every second morning)
